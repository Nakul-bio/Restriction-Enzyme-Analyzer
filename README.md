# Restriction Enzyme Analyzer

A Python tool that identifies restriction enzyme recognition sites in DNA sequences.

## Features
- Detects EcoRI, BamHI, HindIII, and PstI sites
- Finds multiple occurrences of the same site
- Reports positions of matches

## Example

Input:
AAGAATTCGCGCGGATCC

Output:
EcoRI site found at position 3
BamHI site found at position 13

## Technologies
- Python
